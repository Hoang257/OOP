Контейнерный класс должен:

1) хранить множество объектов 

2) управлять добавлением и удалением 

3) предоставлять доступ к объектам

4) позволять итерироваться по коллекции

Для начала создал коллекцию атлетов. Создал объекты класса Athlete (импортировал из lab01) и добавил в наш контейнер.

какие действия или же методы мы сможем сделать с нашей коллекцией ?

1. добавление/удаление, также добавим возможность это сделать по индексу

валидация: нельзя добавить атлета с одним и тем же id, можно только объекты класса Athelete, нельзя удалять спортсмена, который не существует

2. сортировки: по возрасту, по весу, по имени

3. нахождение атлета по id, имени. 

валидация: если не находит атлате, то выдаем ошибку

4. pабота с коллекцией: возвращает сколько спортсменов в коллекции, позволяет итерироваться по коллекции, позволяет обращаться по индексу, тоесть типа среза

5. логические операции над коллекцией: получение только активных атлетов, получение только атлетов определенной категории, получение всех сущевстующих id.

6. вывод самой коллекции.
